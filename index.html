Mini World Tools

Bộ công cụ hỗ trợ tạo script xây dựng tự động trong Mini World

Auto 2D

Công cụ chuyển đổi hình ảnh 2D thành script xây dựng tự động trong Mini World. Tạo các công trình từ hình vẽ, bản đồ hoặc logo,...

  • Hỗ trợ định dạng PNG
  • Chuyển đổi hình ảnh thành mã Lua
  • Có tối ưu màu các khối
  • Xem HDSD trước khi dùng

Auto 3D

Công cụ chuyển đổi mô hình 3D thành script xây dựng trong Mini World. Tạo các công trình từ file 3D(thật ra chỉ có obj) chỉ với vài bước.

  • Nhập file từ Drububu Voxelizer
  • Chuyển đổi JSON thành mã Lua
  • Sử dụng tương tự như 2D
  • Thêm dòng này cho cân xứng với bên kia
